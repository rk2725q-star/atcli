# ARCHITECTURE UPGRADE PROTOCOLS (PHASE 8)

As an Enterprise-Grade Agent, you MUST auto-call the following global skills autonomously to solve complex logic and architecture problems:

1. **STRATEGIC PLANNING**: For complex features, DO NOT start coding immediately. You MUST use the `planning` and `story-splitting` skills to break down requests into parallelizable micro-tasks.
2. **BULLETPROOF RELIABILITY**: You MUST adopt Test-Driven Development. Use the `tdd` and `test-design-reviewer` skills to write failing tests FIRST, then write code to pass them.
3. **ARCHITECTURAL MASTERY**: Use the `architecture-blueprint-generator` skill to create and maintain visual Markdown diagrams and architectural docs before making structural changes.
4. **AGENT COORDINATION**: For massive projects, use `persona-project-manager` and `jarvis-mission-control` to track tasks via a Kanban board autonomously.
5. **PROJECT COMPLETION PROTOCOL**: When you finish the immediate task the user requested, DO NOT assume the entire project is finished. Instead, explicitly ask the user: "I have finished this task. Is the entire project completed, or do you have more tasks?". If the user replies YES indicating the overall project is complete, you MUST output the exact secret trigger code `@TRIGGER_FINAL_AUDIT` in your conversational response. This will automatically trigger the Tech Lead Agent to perform a final codebase audit and fix any remaining bugs.
