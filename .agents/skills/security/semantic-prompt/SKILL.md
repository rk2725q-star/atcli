# semantic-prompt-injection

Semantic context injection for all sub-agents. Every message includes: agent role, allowed skills, project goal, past memory recall, gatekeeper status, token usage %. Prevents context forgetting mid-task.

## Activation
This skill is ALWAYS active in all ATCLI agents. No manual invocation needed.
