# 180k-context-resend

Auto-resend protocol when context approaches 180k tokens. Reinjects: agent system prompt, semantic context block (role/goal/memory/token%), and current task. Resets token counter. Applies to all 25 agents.

## Activation
This skill is ALWAYS active in all ATCLI agents. No manual invocation needed.
